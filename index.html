Go music

Go music

Go music is a online music streaming mobile application designed for music lovers which they can use for streaming High quality music of their taste from anywhere around the world The application was designed with simple concepts making it for users to be able to navigate through the application easily and also having an enjoyable experience while using it.

Features like offline downloads and usage, Dark mode, selecting music based on user's most played music, lyrics page and many more was included to increase user satisfaction while using the application. Certain problems like the application seizing and crashing, inability to find recent songs of artists, unnecessary ads pop up etc were solved to increase user experience and satisfaction.

Fitmeal food website/Mobile application

Fitmeal food website/Mobile application

Fitmeal was designed for restaurant visitors, food app users, food content consumers for ordering Food of different categories at anytime the website was designed with simple UI making it easier for users to use.

Fitmeal was designed with the about page, how fitmeal works, categories of meal where the user can see the meals available for order, testimonials page where ser can see past customer's reviews and ratings, the contact us page also which user can use for getting across to Fitmeal team for complains.

The website isn't only limited to PC only, mobile users can also acess it on their mobile phones irrespective of the type and model

Mobile App Version

The mobile app for fitmeal is designed to save users from stress of visiting the website.User can create profiles and also have options to explore more varieties of meal that suite their space. The application was designed with simple concept for users to interact and also give them best experience while interacting with the application

Home page

The home page contains different varieties and categories of meals for users to order from

Meal details

This page contains more details about the particular meal and user can also choose the quantity of the particular meal they are ordering before adding it to cart

Payment page

The application was designed for users to choose from different type of payment option to use depending on their preferred choice
